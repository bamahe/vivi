# AEO Weekly Loop — ViVi Property Management

## Purpose
Weekly checklist to maintain and improve AI Answer Engine Optimization (AEO) performance for vivipm.com. Run every Monday.

## Weekly Checklist

### 1. Google Search Console (GSC) Review
- [ ] Check **Search Performance** for query impressions and clicks
- [ ] Look for property management queries where vivipm.com appears
- [ ] Note any new queries triggering impressions (expansion opportunities)
- [ ] Check **Coverage** for indexing errors or warnings
- [ ] Verify sitemap is being crawled (last read date)

### 2. AI Answer Engine Spot Checks
- [ ] Search "best property management Tampa Bay" on Google — check for AI Overview
- [ ] Ask ChatGPT "Who offers property management in Tampa Bay?" — check if ViVi appears
- [ ] Ask Perplexity "property management companies Hillsborough County FL" — check citations
- [ ] Ask Claude "Who is Barrett Henry REMAX?" — check entity recognition

### 3. Schema Validation
- [ ] Run homepage through Google Rich Results Test (https://search.google.com/test/rich-results)
- [ ] Run /about through Rich Results Test
- [ ] Fix any schema errors or warnings

### 4. Content Freshness
- [ ] Update any outdated stats or pricing if changed
- [ ] Add 1-2 new FAQ items if common questions emerged that week
- [ ] Review QuickAnswer blocks for accuracy

### 5. Crawl Access
- [ ] Verify robots.txt allows AI crawlers (GPTBot, ClaudeBot, PerplexityBot)
- [ ] Check llms.txt is accessible at vivipm.com/llms.txt
- [ ] Confirm sitemap.xml returns all pages

### 6. Competitor Check
- [ ] Search top 3 competitor names + "property management Tampa" — compare AI visibility
- [ ] Note any competitor schema improvements to match or exceed
